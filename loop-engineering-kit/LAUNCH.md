# Launch Checklist

## Repository
- [ ] Replace `yourusername` links in README
- [ ] Review claims and label illustrative metrics
- [ ] Run tests in a clean environment
- [ ] Enable GitHub Actions and branch protection
- [ ] Configure issue templates and Discussions
- [ ] Add screenshots or diagrams

## Release
- [ ] Create GitHub repository and upload this ZIP
- [ ] Confirm license and contribution policy
- [ ] Tag `v0.1.0` and publish release notes
- [ ] Announce in relevant communities without spamming
- [ ] Track issues, documentation gaps, and user feedback

## Production safety
- [ ] Add budgets, timeouts, and maximum iterations
- [ ] Scrub secrets and personal data from logs
- [ ] Test failure, fallback, and escalation paths
- [ ] Benchmark on representative workloads
